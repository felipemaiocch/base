Como ficou separado

  • O chat publico nao recebe a base nem a chave da IA no navegador.
  • O backend busca a base no Neon a cada requisicao de chat.
  • Somente esta tela autenticada consegue ler e editar o conteudo.
  • A autenticacao usa cookie HTTP-only, entao o JavaScript do navegador nao acessa a sessao.
